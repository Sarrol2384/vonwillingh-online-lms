## 📝 Lesson Learned

**ALWAYS create a manual backup before ANY data modification:**

```sql

SELECT * FROM modules 
WHERE id IN (SELECT id FROM modules WHERE ...);

-- Then do the update
UPDATE modules SET ...

-- If something goes wrong, restore:

SET content = mb.content
FROM modules_backup_20250221 mb
WHERE m.id = mb.id;
```
