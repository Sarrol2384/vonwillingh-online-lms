## 📊 Import Modes

### 1. Create New (Recommended for First Import)
- Creates a brand new course
- Assigns new course ID
- Imports all modules
- Will fail if course with same name already exists

### 2. Update (For Replacing Content)
- Replaces existing course modules
- Keeps course ID
- Deletes old modules, imports new ones
- Use when you have updated course content

### 3. Append (For Adding Modules)
- Keeps existing modules
- Adds new modules to the end
- Useful for adding additional content later
